# 远程规则文件
国内直连（China）、国际加速（Proxy）、广告拦截（Reject）三大基本规则rule文件

1、感谢大神们的付出，每日自动更新。三大基础规则，满足一切需求。

2、规则list文件一般通用于Quantumultx、Loon、Stash、Shadowrocket等，部分软件语法用法略用差异。
